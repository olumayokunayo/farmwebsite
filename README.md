# Admof Farms Website

The official website for **Admof Farms**, a purpose-driven livestock and agribusiness based in Ibadan, Nigeria.  
The platform presents the farm’s operations, services, and values through a modern, interactive, and performance-focused web experience.

## Project Overview

Admof Farms operates across:
- Broiler breeder operations
- Day-old chick sales
- Frozen chicken production
- Ram and goat fattening
- Community-focused, sustainable farming initiatives

This website was developed to establish a strong digital presence, improve customer engagement, and present the brand in a professional and scalable way.

## Key Features

- Modern, responsive UI built with **Next.js and React**
- Component-driven architecture for scalability and maintainability
- Interactive UI elements using **Radix UI**
- Smooth animations and transitions with **Framer Motion**
- Dark/light theme support using **next-themes**
- 3D visual elements using **Three.js** and **@react-three/fiber**
- Accessible, form-driven interactions using **React Hook Form** and **Zod**
- Optimised layout and performance using Next.js rendering capabilities
